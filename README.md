# bs-errors

Utilities for working with errors in bucklescript

## Installation

Install the project:

`npm install bs-errors --save`

And add the dependency to your bs-dependencies in `bsconfig.json`:

```json
"bs-dependencies": [
  "bs-errors"
]
```

The project will be available under the `BsErrors` namespace
